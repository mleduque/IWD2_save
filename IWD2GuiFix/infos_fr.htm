IWD2 GUI Fix | Icewind Dale II — Amélioration des interfaces pour Widescreen
v.1.04c - novembre 2020

IWD2 GUI Fix modifie l'interface principal d'Icewind Dale 2. Il vient en complément du mod Widescreen développé par the bigg, et nécessite donc d'être installé après lui.
Certains problèmes d'affichage existants avec Widescreen, comme par exemple le mauvais affichage de l'interface du jeu lorsqu'il est réduit à la seule barre des PJ, ne sont pas corrigés par ce mod.

Les fichiers de l'interface ont été retravaillés afin qu'ils soient correctement centrés, sans zones noires. Seules certaines résolutions sont actuellement disponibles :

  • 1280x720
  • 1280x800
  • 1280x960
  • 1280x1024
  • 1360x768 | 1366x768 *
  • 1440x900
  • 1600x900
  • 1600x1200
  • 1680x1050
  • 1920x1080
  • 1920x1200

Si vous voulez utiliser une résolution de 1366x768, utilisez ces dimensions lors de l'installation de Widescreen, puis utilisez les fichiers de la résolution 1360x768, ils fonctionneront très bien dans le jeu.

Installation

sous Windows

J'ai joint un fichier setup.exe qui permet de faire une installation rapide. Le programme d'installation remplace automatiquement les fichiers suivants...

    GACTN118.mos
    COMM118.mos 
    GCOMM128.mos 
    GMWDMMB0.mos 
    GMWDSLB0.mos 
    GUCONTB0.mos 
    GUDEATH0.mos 
    GUWBTP30.mos 

    GUIW10.chu
...qui se trouvent dans le dossier \Override d'IWD2, selon la résolution que vous aurez choisie.
  • — une copie de ces fichiers est créée dans le dossier \GUIFix\backup d'Icewind Dale II.
  • — une entrée dans le registre [HKLM\Software\Wow6432Node\IWD2 GUIFix], est créée, afin de stocker le répertoire d'installation et la version installée.

sous Linux | Mac OS

L'installation se fait manuellement, désolé !
  • si vous souhaitez faire une copie de sauvegarde, copiez les fichiers indiqués ci-dessus, présents dans le dossier override d'IWD2 vers un emplacement de votre choix.
  • copiez les fichiers .mos contenus dans le dossier \mos\[sous dossier correspondant à la largeur de la résolution] vers le dossier override du jeu.
  • copiez le fichier GUIW10.chu contenu dans le dossier \chu\[sous-dossier correspondant à la résolution] toujours vers le dossier override du jeu.

Désinstallation

Pour désinstaller automatiquement le mod ou installer une nouvelle résolution, il suffit de lancer à nouveau le setup.exe. Les fichiers copiés dans le backup sont restaurés, et les entrées dans le registre supprimées.
Pour une désinstallation manuelle, il suffit de recopier les fichiers dont vous aurez fait une sauvegarde dans un répertoire à part vers ceux du dossier Override et de les écraser.

Problèmes connus / non-résolus

La meilleure démarche à suivre, avant d'installer IWD2 GUI Fix, est de s'assurer que tout fonctionne déjà bien avec Widescreen : installez Widescreen dans la résolution de votre choix, et assurez-vous que les différents écrans d'interface fonctionnent bien dans le jeu. Si tout marche bien, installez ensuite le mod.

Pour autant, certains problèmes inhérents à Widescreen ne sont pas corrigés par ce mod :

1. Certaines résolutions sont incompatibles avec des formats d'écran, cela tient aux contraintes de Widescreen et/ou du moteur du jeu, que le IWD2 GUI Fix soit installé ou non. Dans ce cas, le jeu ne se lancera tout simplement pas !
Résolutions concernées :

  • 1280 x 854 (3/2) sur un écran 16/9
  • 1600 x 1200 (4/3) sur un écran 16/9
  • 1920 x 1200 (16/10) sur un écran 16/9
Si vous indiquez une résolution dans Widescreen et que le jeu ne se lance pas, essayez-en une autre dans un premier temps, parmis celles qui sont adaptées à votre écran.

2. Dans la plupart des résolutions, un problème d'affichage se produit avec l'interface principal du jeu, lorsqu'on le réduit (touche Y par défaut) au seul bandeau des icones des PJ: une zone figée apparait, de hauteur variable selon la résolution utilisée, totalement disgracieuse, et qui empêche de jouer. Il faut simplement se résoudre à ne pas utiliser l'interface dans cette disposition...
Ce problème vient de Widescreen, je n'ai pas réussi à le contourner. Je l'ai évoqué sur le forum depuis quelque temps déjà, mais sans aucun retour.
Résolutions concernées : toutes sauf 1360x768 et 1280x720 (dans ce dernier cas, le bandeau n'apparait pas, le problème est réglé !)

3. Lorsqu'on lance un jeu en réseau, le jeu passe automatiquement en mode fenêtré. Il suffit d'appuyer sur ALT + Enter pour passer successivement en mode fenêtré /mode plein écran

Crédits / Remerciements

Ce mod n'est ni développé ni avalisé par Interplay, Black Isle ou Bioware.
Il a été créé dans le seul but d'améliorer le plaisir de jouer à Icewind Dale 2. Il ne réalise ni ne cherche à réaliser aucun profit. Vous pouvez le distribuer librement, en respectant ce principe, et avec l'accord de son auteur, Luren.

Merci à the bigg pour son excellent mod Widescreen, qui permet de jouer à Baldur's Gate et Icewind Dale sur les "nouveaux" écrans 16/9 et 16/10 !
