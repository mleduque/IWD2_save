Icewind Dale II GUI fix for Widescreen
version 1.04c - november 2021

This mod changes the main GUI of Icewind Dale 2. It works with the Widescreen * mod developed by the bigg, and therefore needs to be installed after him.
Some existing display problems knowns with Widescreen, such as the bugged drop away interface panels of the main interface, are not fixed by this mod.

I reworked the GUI files of the main interface so that they are properly centered, without black areas. The most common resolutions are currently available.

  • 1280x720
  • 1280x800
  • 1280x960
  • 1280x1024
  • 1360x768
  • 1440x900
  • 1600x900
  • 1600x1200
  • 1680x1050
  • 1920x1080
  • 1920x1200

Installation

This installation setup automatically replaces the following files :

    GACTN118.mos
    COMM118.mos 
    GCOMM128.mos 
    GMWDMMB0.mos 
    GMWDSLB0.mos 
    GUCONTB0.mos 
    GUDEATH0.mos 
    GUWBTP30.mos 
    Guiw10.chu
...located in the IWD2 \Override folder, according to the resolution you choose.
  • — A copy of these files is created at the path \GUIFix\backup of Icewind Dale II.
  • — An entry in the registry [HKLM\Software\Wow6432Node\IWD2 GUIFix], is created to store the installation directory and the installed version.

Uninstall

To automatically uninstall the mod or install a new resolution, simply run the setup.exe again. The files copied into the backup folder are restored, and the entries in the registry deleted.

Knowns / Unsolved problems

The best thing to do, before installing this mod, is to make sure that everything is allright with Widescreen: install Widescreen in the resolution of your choice, and make sure that the various interface panels work well in the game. If everything works fine then install my mod.

However, certain problems inherent to Widescreen are not corrected by this mod:
1.Some resolutions are not compatible with some screen formats, this is due to the constraints of Widescreen and maybe the game engine, whether the IWD2 GUI Fix is installed or not. In this case, the game will simply not launch !
Resolutions concerned :
  • 1280 x 854 (3/2) with a 16/9 screen
  • 1600 x 1200 (4/3) with a 16/9 screen
  • 1920 x 1200 (16/10) with a 16/9 screen
If you specify a resolution in Widescreen and if the game does not start, try a different one first, among those that are suitable for your screen.

2.In most resolutions (except 1360x768 and 1280x720), a display problem occurs with the main interface of the game, when some panels are hidden (by default, pressing Y key will make the character and action button panel on the main screen drop away) : a frozen area appears, of variable height according to the resolution used, totally unsightly, and which prevents well playing. You just have to resolve not to use the interface as it, and press Y to use another disposition...
This problem comes from Widescreen, I couldn't get around it. I have mentioned it on the forum for some time now, but without any feedback.

3. When you launch a network game, the game automatically switches to windowed mode. Just press ALT + Enter to switch successively to windowed / full screen mode.
