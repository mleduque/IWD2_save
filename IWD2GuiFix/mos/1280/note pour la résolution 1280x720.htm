Installation manuelle d'une résolution 1280x720

Si le fichier guiw10.chu se copie tel quel, Widescreen utilise en revanche, pour la résolution 1280x720, des noms différents pour les fichiers .MOS, .
Lors d'une installation manuelle, il faudra donc prendre les fichiers .MOS du dossier 1280 et les renommer comme suit:

While the guiw10.chu file copies itself as is, Widescreen use different names for a 1280 x 720 resolution. So the .mos files should to be renamed as follows, before being copied to the "Override" folder:

  • GACTN118.mos en GACTN008.mos
  • GCOMM118.mos -> GCOMM0A8.mos
  • GCOMM128.mos -> GCOMM108.mos
  • GMWDMMB0.mos -> GMWDMMB8.mos
  • GMWDSLB0.mos -> GMWDSLB8.mos
  • GUCONTB0.mos -> GUCONTB8.mos
  • GUDEATH0.mos -> GUDEATH8.mos
  • GUWBTP30.mos -> GUWBTP38.mos :
